# notmess.ru

Официальный лендинг [NotMess](https://notmess.ru) — защищённого мессенджера со сквозным шифрованием.

## О проекте

NotMess — веб-мессенджер в стиле Telegram с авторизацией через Telegram бота, сквозным шифрованием (E2E), системой звёзд и подарков.

- **Лендинг**: https://notmess.ru
- **Веб-версия**: https://web.notmess.ru
- **Telegram канал**: https://t.me/notmesstg
- **Telegram бот**: https://t.me/notmess_autobot

## Технологии

- HTML, CSS (vanilla)
- GitHub Pages
